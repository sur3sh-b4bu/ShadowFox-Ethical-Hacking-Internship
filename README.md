# ShadowFox Ethical Hacking Internship

This repository contains weekly reports submitted as part of the ShadowFox Ethical Hacking Internship.

## Internship Focus
- Ethical Hacking Methodologies
- Web Application Security Testing
- Vulnerability Assessment
- Security Reporting

## Repository Structure
- Reports/WEEK-01 – Fundamentals and reconnaissance
- Reports/WEEK-02 – Vulnerability discovery
- Reports/WEEK-03 – Exploitation techniques
- Reports/WEEK-04 – Final report and remediation
